### 六、混动架构技术细节补充

#### **串联混动（Series Hybrid）技术细节**
```
原动机（涡轴/涡桨/活塞）→ 发电机 → 整流器 → 直流母线 → 逆变器 → 电动机 → 推进器
                                      ↓
                                电池组（功率补偿/能量回收/应急备用）
```

**能量流特点**：
- 机械能→电能→机械能，两次能量转换损失约10-15%
- 发动机与推进器完全解耦，可分别优化效率
- 电池提供峰值功率补偿，降低发动机额定功率需求

#### **并联混动（Parallel Hybrid）技术细节**
```
原动机 → 离合器/耦合机构 → 减速器 → 螺旋桨/风扇
              ↓
           电动机 ← 逆变器 ← 电池
```

**能量流特点**：
- 机械能直接传递为主，电机辅助路径为辅
- 起飞/爬升：发动机+电机共同输出
- 巡航：发动机单独工作，电机可发电充电
- 下降：电机作为发电机回收能量

#### **涡轮电动分布式推进（TEDP）技术细节**
```
原动机（1-2台）→ 发电机 → 高压配电系统 → 多台分布式电动机/风扇
                    ↓
              电池/超级电容（功率平滑）
```

**关键技术挑战**：
- 兆瓦级电力系统热管理（液冷/风冷）
- 高压配电（750V-7500V）绝缘与安全
- 超导电机技术（针对N3-X等50MW级系统）
- 功率电子器件重量与效率平衡

**当前技术成熟度**：
- **串联混动**：地面测试完成→飞行验证前（TRL 6-7）
- **并联混动**：验证机研制→飞行测试（TRL 5-6）
- **涡轮电动分布式**：概念验证→地面测试（TRL 4-5）
